## Getting Started (5 Minutes)

### Step 1: Create Your First Tags

1. Click the **"+ Manage Tags"** button (emerald green, top-right of page)
2. In the modal that opens:
   - Enter a tag name (e.g., "Work")
   - Click the color picker to choose a color (or type a hex code like `#10B981`)
   - Click **"Create Tag"**
3. Repeat for more tags (Personal, Urgent, etc.)
4. Click **"Close"** when done

**ğŸ’¡ Pro Tip**: Choose distinct colors so you can spot tags at a glance!

### Step 2: Apply Tags to a Todo

**When Creating a New Todo**:
1. Fill out the todo form as usual (title, due date, priority)
2. Below the form, you'll see your tags as pill buttons
3. Click a tag to select it (it will fill with color and show a checkmark âœ“)
4. Click multiple tags to apply several at once
5. Click **"Add"** to create the todo

**When Editing an Existing Todo**:
1. Click **"Edit"** on any todo
2. Scroll down to see the tags section
3. Click tags to toggle them on/off
4. Click **"Save"** to apply changes

### Step 3: Filter by Tags

1. Look for the **"All Tags"** dropdown (next to "All Priorities")
2. Select a tag name to filter
3. Only todos with that tag will appear
4. Select "All Tags" again to see everything

## Troubleshooting

### "Tag name already exists"

**Problem**: You tried to create a tag with the same name as an existing one.

**Solution**: 
- Choose a different name (e.g., "Work Projects" instead of "Work")
- Or edit the existing tag instead

### Tags not showing on todo

**Problem**: You selected tags but they don't appear after creating todo.

**Solution**:
- Make sure you clicked "Add" after selecting tags
- Check if tags appear in the modal (may be a display issue)
- Try refreshing the page

### Tag filter not working

**Problem**: Selecting a tag doesn't filter the list.

**Solution**:
- Clear other filters first (search, priority)
- Make sure some todos actually have that tag
- Try selecting "All Tags" then the tag again

### Can't delete tag

**Problem**: Delete button doesn't work or tag reappears.

**Solution**:
- Make sure you clicked "Delete" not "Edit"
- Confirm the deletion dialog if it appears
- Refresh the page if tag still shows

## Frequently Asked Questions

### Can I have duplicate tag names?
No, tag names must be unique per user (case-insensitive).

### How many tags can I create?
Unlimited! But we recommend 10-15 for optimal organization.

### How many tags can I apply to one todo?
As many as you want, but 1-3 is recommended for clarity.

### Do tags work with recurring todos?
Yes! When a recurring todo is completed, the next instance keeps the same tags.

### Can I change a tag's color after creating it?
Yes! Edit the tag in the modal and all todos update instantly.

### What happens if I delete a tag?
The tag is removed from all todos that have it (CASCADE delete).

### Can I search by tag?
Yes! Use the "All Tags" dropdown to filter, or combine with text search.

### Do tags export with todos?
Currently tags are not included in JSON/CSV exports (coming in future update).

### Can I share tags with other users?
No, tags are user-specific. Each user has their own tag library.

### Are there keyboard shortcuts?
Not yet, but coming in a future update!
